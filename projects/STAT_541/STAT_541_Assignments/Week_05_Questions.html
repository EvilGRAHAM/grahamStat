4.19

logit_reg_4.19 <- function(Gender, Religion, Political){
  log_odds <-
    -0.11 + 
    if_else(Gender == "M", 0.0, 0.16) +
    if_else(Religion == "J", 0.0, if_else(Religion == "C", -0.66, -0.57)) +
    if_else(Political == "I", 0.0, if_else(Political == "R", -1.67, 0.84))
  log_odds
}

a.

\[ OR_{\text{Gender}} = e^{\hat{\beta}_{1}^{G} - \hat{\beta}_{2}^{G}} = e^{0.16 - 0} = 1.1735 \] Holding all else equal, the odds of supporting legalized abortion increases on average by a multiplicative factor of 1.1735 for females vs. males.

b.

c.

\[ \hat{\beta}_{2}^{G} = 0.16 \implies OR_{\text{Gender}} = e^{\hat{\beta}_{1}^{G} - \hat{\beta}_{2}^{G}} = e^{0 - 0.16} = 0.8521 \] Holding all else equal, the odds of supporting legalized abortion decreases on average by a multiplicative factor of 0.8521 for males vs. females.

d.

\[ \hat{\beta}_{1}^{G} + \hat{\beta}_{2}^{G} = 0, \hat{\beta}_{1}^{G} - \hat{\beta}_{2}^{G} = 0.16 \implies \hat{\beta}_{1}^{G} = 0.08, \hat{\beta}_{2}^{G} = -0.08 \implies \] \[ OR_{\text{Gender}} = e^{0.08-(-0.08)} = e^{0.16} = 1.1735 \]

\[ \hat{\beta}_{\text{Gender}} = \frac{\ln 1.23 + \ln 12.88}{2} = 1.3813 \approx 1.38 \neq \ln 1.38 = 0.3221 \implies \] \[ OR_{\text{Gender}} = e^{1.38} = 3.9803 \approx 3.98 \] Therefore, if the confidence interval is correct the log odds ratio for gender must be 1.38, giving an estimated odds ratio of 3.98.
